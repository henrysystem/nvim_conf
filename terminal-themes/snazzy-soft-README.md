# Snazzy Soft - Versión con colores suavizados para OpenCode

Variante de Snazzy con colores menos saturados para contexto MCP/LSP.

## Diferencias vs Snazzy original

| Color | Snazzy | Snazzy Soft |
|-------|--------|-------------|
| Cyan | `#9AEDFE` (muy brillante) | `#7AC7E0` (suave) |
| Magenta | `#FF6AC1` (muy brillante) | `#D88AB8` (suave) |
| Yellow | `#F3F99D` (mantiene) | `#F3F99D` (mantiene) |
| Bright Cyan | `#9AEDFE` | `#6BB5D0` |
| Bright Magenta | `#FF6AC1` | `#C878A8` |

Los demás colores se mantienen igual.

## Instalación

Igual que Snazzy original pero usando los archivos `snazzy-soft-*`
